3rd Person Controller

This is a 3rd person camera setup where you can control movement and rotation using the keyboard.

Controls

  • W ↑
    Move Forward
  • A ←
    Turn Left
  • S ↓
    Move Backward
  • D →
    Turn Right
  • Shift
    Run
